# bank-simple
A simple bank framework using OOP - clients, accounts and verification

Version 1.0 (old) - only the basic of client and account information available. Portuguese language, pending translation.

Version 2.0 (old) - now working clients, accounts, bank information and deposit and withdrawn commands. English language. 

Version 3.0 (old) - authentication method implemented.

Version 4.0 (old) - user interface implemented, now the program remember the amount of money deposited and withdrawn. Last step is implementing save and load features.

Version 5.0 (current) - save command implemented.
